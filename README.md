#  Molecule Ideation Assistant
Powered by NVIDIA BioNeMo + Llama 3.3 70B

## What it does
AI pipeline that generates optimised drug analog candidates 
from a lead compound under potency, toxicity, and synthesis constraints.

## Tech Stack
- NVIDIA BioNeMo (MolMIM) — analog generation
- Llama 3.3 70B (NVIDIA NIM) — reasoning & ranking  
- RDKit — molecular validation & scoring
- ChEMBL — real FDA-approved lead compounds

## Results
- 10 analogs generated per lead compound
- Ideation time: weeks → under 5 minutes
